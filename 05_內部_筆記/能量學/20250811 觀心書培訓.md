### 觀心書重點

3. 行動方案要具體
4. 參解觀心書 - 是不是有慣性沒有發現，沒有過關


### 為什麼要寫行動方案
1. 無法建設人生的學習，都是負擔
怎麼說？二階學到了怎麼參解，認真的寫了一年，人生沒有很大的變化。
因為寫觀心書的時候沒有寫行動方案
根據自己的狀態，給自己一個可以實現的行動方案
以可以調整行為的行動方案為主，行為無法調整的行動方案就不夠落地
不要陷入完美主義
可以從調整自己的行為開始，借假修真
2. 知行合一
陰陽為一組，一陰接一陽，這樣合一的情況下，會達成心想事成的狀態
心想為陰，事成爲陽
為什麽有些人不能做大事，就是因為他不能從小事開始做，因為想完了小事，又一直想，一直想，沒時間做大事
3. 創造心印，信念轉化爲行動

發現自己的行動方案越累積越多，代表給自己的行動方案太難了

### 有效的行動方案
**兩個方向**
1. 減少自己種的壞種子
有人知道為什麼是減少而不是停止嗎？
要會給自己架階梯
一個月只改掉一個壞習慣
在座的各位覺得自己身上有沒有 100 個壞習慣
如果真的做不到，給自己一個彌補方案(一種架設階梯)
控制伴侶半年
2. 種好種子
4W1H(what, who, where, when, how)
何事，何人，哪裡，何時，如何


### 行動方案的誤區
1. 勿做終生承諾 -> 不要說再也不
2. 行動方案太多了 -> 會讓達成率變低，達成率變成批判自己的原因
3. 不落地 -> 如何落地？+, - 的字數要變多才入心，執行上次沒執行好的行動方案
4. -+ : 合在一起寫，不建議，因為我們要覺察自己的起心動念，分開來寫比較好覺察

### 養成習慣的成功關鍵
1. 每日 TODO 一定要完成
2. 走道命，對自己狠一點
對自己狠一點，老天爺會寬待你
主動去面對自己的苦，那就不會吃被動的苦
3. TODO 一直無法完成 -> 求救輔導員

## 21 天觀心書練習

**重複的錯要深度覺察**：

- 做 "痛的參解" 來幫助穿越
